Setup

  • Terraform – Proxmox VM’er (1 control plane + 2 workers), statisk IP, VLAN 551.
  • Ansible – Docker + denne app på alle noder; nginx som load balancer på control plane.
  • Docker – Node.js + Express i container; docker-compose.
  • Du rammer control plane (10.133.51.120) → nginx round-robin til worker 1 og worker 2.
